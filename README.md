# Bioinformatics Programming Foundations: R & Python 💻

This repository contains my core programming assessments from the MSc Cancer Genomics program at QMUL. It demonstrates a solid foundation in algorithmic thinking, data manipulation, and software development tailored for biological datasets.

## 🎯 Key Competencies

### 📊 R Programming (Biomedical Data Analysis)
- **Data Wrangling:** Loading and processing `TSV/CSV` files, managing row names, and calculating summary statistics (mean, median, SD).
- **Advanced Data Types:** Working with `ExpressionSet` classes and S4 objects for genomic data.
- **Visualisation:** Creating complex plots with `ggplot2`, including boxplots with statistical significance and regression analysis.
- **Custom Functions:** Developing robust R functions for DNA sequence manipulation (Transcription, Translation, Substitution).

### 🐍 Python Programming (Genomics & OOP)
- **Genomic Algorithms:** Designing Python functions to calculate nucleotide composition and sequence properties.
- **Object-Oriented Programming (OOP):** Modeling biological entities (like Cells and Genes) using Python Classes and methods.
- **Data Science Tools:** Using `Pandas` for filtering and analyzing TCGA liver cancer datasets and `Matplotlib` for visualising gene expression trends.
- **Regex & Logic:** Applying regular expressions and conditional logic to solve sequence-based problems.
